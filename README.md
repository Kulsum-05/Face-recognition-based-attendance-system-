# Face Recognition Based Attendance System

## 📌 Project Overview
This project is an automated attendance system using face recognition. It replaces manual attendance marking with real-time face detection using a webcam.

## 🚀 Features
- Real-time face detection
- Automatic attendance marking
- Stores attendance with timestamp
- Simple and efficient system

## 🛠️ Technologies Used
- Python
- OpenCV
- face_recognition library
- NumPy
- CSV for data storage

## 📷 How it works
1. Load known face images
2. Capture live webcam feed
3. Match faces with known encodings
4. Mark attendance in CSV file

## 📂 Output
Attendance is stored in `attendance.csv` with name and time.
